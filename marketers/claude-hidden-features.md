## 01 — Projects: შენი workspace ბრენდის voice-ით

**რა არის:** Claude Project არის სამუშაო სივრცე, სადაც ერთხელ ჩაგდები შენი ბრენდის voice-ს, აუდიტორიასა და პროდუქტებს. ყოველ ჩათში, რომელიც ამ Project-ში იქმნება, Claude უკვე ცნობს შენს კონტექსტს.

**რატომ ღირს:** აღარ დაგჭირდება ყოველ ჩათში გამეორება — „შენ ხარ ქართული ბრენდის მარკეტერი, ჩვენი ბიზნესი არის X, ჩვენი აუდიტორია არის Y..."

**პრაქტიკული გამოყენება:**
- ერთხელ setup → 100+ ჩათში ცნობილი ბრენდი
- ერთი Project თითოეული კლიენტისთვის (თუ ფრილანსერი ხარ)
- სხვადასხვა Project — თუ რამდენიმე ბრენდი გაქვს

**ხელმისაწვდომია:** Claude Pro, Max, Team, Enterprise

> 💡 **სრული setup-ის ინსტრუქცია** იხილე — [Claude Project Setup გაიდი](./claude-project-setup.md)

## 02 — Artifacts: ცოცხალი დოკუმენტები

**რა არის:** Artifact არის HTML, კოდი, SVG ან document, რომელიც Claude აშენებს ჩათის გვერდით ცალკე panel-ში — შეგიძლია ნახო, შეცვალო და გადმოწერო რეალურ დროში.

**რატომ ღირს:** ტექსტიდან მზა output — pitch deck, landing page, calculator — ყველაფერი იქმნება ცოცხლად, ცალკე ხელსაწყოების გარეშე.

**პრაქტიკული გამოყენება:**
- პრომპტი: „გამიკეთე quiz audience-ისთვის" → მზა web-app 2 წუთში
- პრომპტი: „გადააქციე ეს ცხრილი ინტერაქტიულ dashboard-ად" → ფუნქციური HTML
- პრომპტი: „დაწერე landing page ახალი პროდუქტისთვის" → მზა page, რედაქტირებადი

**ხელმისაწვდომია:** Claude.ai-ის ყველა მომხმარებელი (free-შიც, შეზღუდვებით)

## 03 — Custom Styles: ბრენდის ხმა შენახული ერთხელ

**რა არის:** Custom Style არის შენახული writing voice. ერთხელ აღწერ ბრენდის ტონს — მერე ნებისმიერ ჩათში ერთი ღილაკით ჩაერთვება.

**რატომ ღირს:** აღარ დაგჭირდება ყოველ ჯერზე გამეორება — „დაწერე უფრო თბილად, მოკლედ, ქართულად, ჩემი ბრენდის ტონით".

**პრაქტიკული გამოყენება:**
- 3 brand voice (formal, casual, fun) — 1 ღილაკით ცვლი
- ერთი style კლიენტის ბრენდისთვის, მეორე — შენი პერსონალური LinkedIn-ისთვის
- გადაამოწმე — Claude შენი ხმით წერს, არა generic AI-ის ტონით

**ხელმისაწვდომია:** Claude Pro, Max, Team, Enterprise

## 04 — Chat Search: შენი ისტორია — ბიბლიოთეკა

**რა არის:** ძიება ყველა შენს წინა ჩათში — keyword-ით, თემით, თარიღით.

**რატომ ღირს:** ის prompt, რომელიც 3 კვირის წინ მაგრად იმუშავა — 3 წამში პოულობ.

**პრაქტიკული გამოყენება:**
- ძიება „meta description" → 12 ვარიანტი, ყველა შენი წინა მუშაობიდან
- ძიება „client X" → ყველა მათ შესახებ ჩათი
- ისტორიის გადახედვა — როგორ ფიქრობდი 6 თვის წინ vs ახლა

**ხელმისაწვდომია:** Claude.ai-ის ყველა მომხმარებელი

## 05 — Voice Mode: ხმით საუბარი მობილურზე

**რა არის:** Claude-ის mobile აპში ხმოვანი რეჟიმი — Claude უსმენს, საუბრობს უკან, ცოცხალ დიალოგში.

**რატომ ღირს:** brainstorm-ი walking-ის დროს, hands-free. 20 წუთიანი გასეირნება გადააქცე 10 ხმოვან იდეად — და სანამ კომპიუტერთან მიდიხარ, უკვე გექნება ფინალური brief.

**პრაქტიკული გამოყენება:**
- ახალი კამპანიის idea-შტურმი დილით ყავის სმის დროს
- პრეზენტაციის გამეორება — ხმით, Claude-ის feedback-ით
- კლიენტის შეხვედრის შემდეგი notes-ის ხმით ჩაწერა

**ხელმისაწვდომია:** Claude mobile აპი (iOS, Android)
