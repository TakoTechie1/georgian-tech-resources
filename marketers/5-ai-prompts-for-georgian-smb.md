## როგორ გამოვიყენო ეს prompt-ები

ყველა prompt-ი მუშაობს როგორც Claude-ში, ისე ChatGPT-ში, Gemini-ში და სხვა მსგავს მოდელებში.

**4 მარტივი ნაბიჯი:**

1. აირჩიე prompt, რომელიც შენს ამოცანას შეესაბამება
2. შეცვალე ფიგურულ ფრჩხილებში `[...]` მოცემული ცვლადები შენი ბრენდის მონაცემებით
3. ჩასვი AI-ში და მიიღე შედეგი
4. გადაამოწმე გრამატიკულად და ბრენდის ხმით — AI ყოველთვის ადამიანის რედაქტირებას საჭიროებს

> 💡 რაც უფრო კონკრეტულია ცვლადი, მით უკეთესი იქნება შედეგი. „ჩვენი პროდუქტი" — ცუდი მაგალითია, „ხელნაკეთი ვერცხლის საყურეები 18-30 წლის ქალებისთვის, ფასი 80-150 ლარი" — სასურველი დონე.

## Prompt #1 — SEO Meta აღწერების Batch Generator

**როდის გამოვიყენო:** როცა 10+ პროდუქტს ან გვერდს უწერია meta description ხელით.

**რამდენ დროს დაზოგავ:** დაახლოებით 30 წუთის სამუშაო — 2-3 წუთში.

```
შენ ხარ SEO სპეციალისტი ქართული ბაზრისთვის.

პროდუქტი/გვერდი: [პროდუქტის ან გვერდის სახელი]
Target keyword: [საძიებო სიტყვა ან ფრაზა]
Target audience: [ვინ ეძებს ამ პროდუქტს]

დაწერე 5 meta description ვარიაცია ქართულად:
- 150-160 სიმბოლო თითოეული
- Target keyword ბუნებრივად 1-ჯერ თითო ვარიანტში
- Call-to-action ფინალში (მაგ. „გაიგე მეტი", „დაჯავშნე")
- ერთი ემოციური სიტყვა შუაში
- ერთი ვარიანტი აუცილებლად კითხვითი ფორმით

შედეგი ჩამოაყალიბე ცხრილად: ვარიანტი | სიმბოლოების რაოდენობა | მთავარი ფოკუსი.
```

## Prompt #2 — Facebook Ad Copy 5 ემოციური Angle-ით

**როდის გამოვიყენო:** ერთი პროდუქტისთვის რეკლამის რამდენიმე ვარიაცია გჭირდება A/B ტესტისთვის.

**რამდენ დროს დაზოგავ:** 1-2 საათი → 5 წუთი.

```
შენ ხარ performance marketer ქართული ბაზრისთვის.

პროდუქტი ან სერვისი: [აღწერე 1-2 წინადადებაში]
სამიზნე აუდიტორია: [ვინ ყიდულობს — ასაკი, ინტერესი, საჭიროება]
მთავარი benefit: [რა გადაწყვეტილებას აძლევ კლიენტს]
Call-to-action: [მაგ. „დაჯავშნე ახლავე", „გაიგე მეტი"]

დაწერე 5 Facebook ad copy ვარიაცია ქართულად, თითო — სხვა ემოციურ angle-ზე:

1. FOMO (დაკარგვის შიში) — შემოთავაზების შეზღუდულობა
2. Social proof — სხვა ხალხის ნდობა და გამოცდილება
3. Curiosity (ცნობისმოყვარეობა) — გაცვლადი კითხვა ან ფაქტი
4. Transformation (ცვლილება) — „იყავი X, გახდი Y"
5. ავტორიტეტი — ექსპერტული ცოდნა და სანდო წყარო

თითოეული max 90 სიმბოლო. CTA ბოლოს. პუნქტუაცია სუფთა ქართული.
```

## Prompt #3 — Google Business Profile-ის SEO გაუმჯობესება

**როდის გამოვიყენო:** Google Maps-ის ძიებაში გვერდი დაბლა იყრება, ან აღწერა ცარიელი/ცუდი გაქვს.

**რამდენ დროს დაზოგავ:** ერთჯერადი 30 წუთიანი ცვლილება — ხანგრძლივი SEO ეფექტი (ლოკალური traffic +20-40%).

```
გადააკეთე ქართული Google Business Profile-ის აღწერა SEO-ისთვის.

არსებული აღწერა: [paste — დაკოპირე შენი ამჟამინდელი ტექსტი]
ქალაქი + რაიონი: [მაგ. თბილისი, ვაკე]
სერვისი ან ნიშა: [მაგ. სილამაზის სალონი, კონდიტერია]
სამიზნე აუდიტორია: [ვინ მოდის შენთან]
უნიკალური განსხვავება: [რა გადაწყვეტილებას აძლევ კლიენტს, რასაც კონკურენტი ვერ აძლევს]

მოთხოვნები:
- Keyword „[ქალაქი + სერვისი]" ბუნებრივად 2-ჯერ
- სიტყვა „ადგილობრივი" ან „სამეზობლო" — 1-ჯერ
- მუშაობის საათები ჩასვი ბუნებრივად
- ცხადი call-to-action ფინალში (დაჯავშნე / გვესტუმრე / დარეკე)
- ტონი — მეგობრული, არა formal
- მაქსიმუმ 750 სიმბოლო

დააბრუნე საბოლოო ვერსია + სიმბოლოების რაოდენობა.
```

## Prompt #4 — Customer Sentiment ანალიზი კომენტარებიდან

**როდის გამოვიყენო:** გჭირდება სწრაფი გაგება — რას ფიქრობს თქვენი აუდიტორია, რა აწუხებს, რა მოსწონს.

**რამდენ დროს დაზოგავ:** Content calendar-ის იდეები 1 საათში.

```
აი 30+ კომენტარი ჩემი Facebook გვერდიდან ან Google review-ებიდან:

[paste — ჩასვი ყველა კომენტარი ერთად, ერთი ცარიელი ხაზით გაყოფილი]

გააანალიზე და დააბრუნე:

1. ტოპ 5 ხშირი წუხილი ან კითხვა — ჩამოწერე priority-ს მიხედვით
2. ტოპ 3 რაში გვაფასებენ ყველაზე ხშირად
3. ემოციური ტონის განაწილება — დადებითი / ნეიტრალური / უარყოფითი (პროცენტებში)
4. 3 კონკრეტული რეკომენდაცია — როგორ გავაუმჯობესოთ პროდუქტი ან სერვისი
5. 1 ახალი content-იდეა — პოსტი, რომელიც პასუხობს ყველაზე ხშირ კითხვას

შედეგი ჩამოაყალიბე ცხადად — bullet point-ებით, არა დიდი აბზაცებით.
```

## Prompt #5 — Content Hook Generator

**როდის გამოვიყენო:** ახალი პოსტი გინდა, მაგრამ პირველი ხაზი არ მოდის.

**რამდენ დროს დაზოგავ:** 10 hook → 10 პოსტი → 2 კვირის content.

```
შენ ხარ social media copywriter ქართული ბაზრისთვის.

თემა ან პროდუქტი: [რაზე უნდა დაიწეროს პოსტი]
პლატფორმა: [Instagram / LinkedIn / Facebook]
ბრენდის ტონი: [მაგ. პროფესიული, მეგობრული, იუმორისტული]
მიზანი: [engagement / sales / awareness / education]

დაწერე 10 hook (პოსტის პირველი ხაზი) ქართულად, თითო 8-12 სიტყვით:

- 3 კითხვითი ფორმით
- 2 cheeky ან მოულოდნელი ფაქტით
- 2 პირადი ისტორიით („წინა კვირას...", „დღეს დილით...")
- 2 contrarian take-ით („ყველა ფიქრობს რომ X, მაგრამ რეალურად Y")
- 1 round-number სტატისტიკით

თითო hook-ის შემდეგ მოკლედ ახსენი — რატომ მუშაობს ეს კონკრეტული formula.
```
