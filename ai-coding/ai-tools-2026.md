## 💬 ჩატბოტები — ტექსტური AI

### 🥇 Claude (Anthropic) — Opus 4.7

**ვებგვერდი:** https://claude.ai

**ძლიერი მხარეები:**
- ყველაზე კარგი წერისთვის და გრძელი ანალიზისთვის
- ყველაზე ნაკლები ჰალუცინაცია (გამოგონილი ფაქტები)
- შესანიშნავი კოდირებაში (განსაკუთრებით Claude Code)
- ეთიკური ქცევა, არ აყვება მაამებლობას
- 200K+ token context window — დიდ დოკუმენტებს კარგად ახერხებს

**სუსტი მხარეები:**
- ქართული ცოტა ნაკლებად სწორი ვიდრე ChatGPT-ის
- სურათების და ვიდეოს გენერაცია არ აქვს
- web search შეზღუდული

**ფასი:** $20/თვე (Pro), $100/თვე (Max)

**როდის გამოვიყენოთ:** გრძელი ტექსტების წერა, კოდის წერა, რთული ანალიზი, რესერჩი

---

### 🥈 ChatGPT (OpenAI) — GPT-5

**ვებგვერდი:** https://chat.openai.com

**ძლიერი მხარეები:**
- ყველაზე ცნობადი ბრენდი
- შესანიშნავი ქართული (ერთ-ერთი საუკეთესო)
- DALL-E ინტეგრაცია (სურათების გენერაცია)
- Code Interpreter / Advanced Data Analysis
- ყველაზე დიდი plugin/GPT ეკოსისტემა

**სუსტი მხარეები:**
- მეტი ჰალუცინაცია ვიდრე Claude-ში
- ხშირად მაამებლობს ("sycophancy" პრობლემა)
- ფასი ყველაზე მაღალი Pro-ზე ($200/თვე)

**ფასი:** $20/თვე (Plus), $200/თვე (Pro)

**როდის გამოვიყენოთ:** ქართული ტექსტი, სურათების გენერაცია, ცხრილების ანალიზი, ბევრი სხვადასხვა task ერთ ადგილას

---

### 🥉 Gemini (Google) — Gemini 2.5

**ვებგვერდი:** https://gemini.google.com

**ძლიერი მხარეები:**
- ყველაზე დიდი context window — 2M+ tokens (1500 გვერდი ერთდროულად!)
- შესანიშნავი Google Workspace-თან (Gmail, Docs, Sheets)
- უფასო ვერსია ბევრად უკეთესი ვიდრე ConkurentEbis
- Multimodal — სურათი, ვიდეო, აუდიო ერთად

**სუსტი მხარეები:**
- ცოტა ნაკლებად "ჭკვიანი" ვიდრე Claude/GPT რთულ ამოცანებში
- ქართული — საშუალო ხარისხი

**ფასი:** უფასო ან $20/თვე (Advanced)

**როდის გამოვიყენოთ:** დიდი დოკუმენტების ანალიზი, Google ეკოსისტემაში მუშაობა, უფასო ვარიანტი

---

### Grok (xAI) — Grok 4.3

**ვებგვერდი:** https://x.com/i/grok

**ძლიერი მხარეები:**
- რეალურ დროში X (Twitter) ინტეგრაცია
- ნაკლები cenzura ვიდრე სხვებს
- კარგი breaking news-ისთვის

**სუსტი მხარეები:**
- ხშირად biased
- ხარისხი ნაკლები ვიდრე Claude/GPT/Gemini

**ფასი:** $8/თვე (X Premium-ში შედის)

**როდის გამოვიყენოთ:** Twitter/X-ის ტრენდები, breaking news ანალიზი

## 🚀 ჩემი პერსონალური Stack 2026

ეს არის ის რასაც ყოველდღე ვიყენებ:

1. **Claude Pro** ($20/თვე) — წერა, კოდი, ანალიზი
2. **Cursor** ($20/თვე) — IDE
3. **Gemini Free** — დიდი დოკუმენტები
4. **ChatGPT Plus** ($20/თვე) — ქართული ტექსტი + სურათები
5. **ElevenLabs Starter** ($5/თვე) — voiceover-ისთვის
6. **Midjourney Basic** ($10/თვე) — სურათები

**სულ:** ~$75/თვე — ჩემი პროდუქტიულობა გაიზარდა მინიმუმ 5-ჯერ.
