## როგორ გამოვიყენო ეს prompt-ები

ყველა prompt იწერება ისე, რომ მუშაობდეს ნებისმიერ AI მოდელთან: ChatGPT, Claude, Gemini, Copilot.

**3 მარტივი ნაბიჯი:**

1. აირჩიე prompt, რომელიც შენს საჭიროებას შეესაბამება
2. შეცვალე ფიგურულ ფრჩხილებში `{...}` მოცემული ცვლადები შენი ინფორმაციით
3. ჩასვი AI-ში და მიიღე შედეგი

**რჩევა:** რაც უფრო კონკრეტული ხარ ცვლადებში, მით უკეთესი იქნება შედეგი. "ბიზნესი" — ცუდი მაგალითია, "ხელნაკეთი კერამიკის ონლაინ მაღაზია თბილისში" — სასურველი დონე.

## Instagram-ის Caption-ები

### 1. Engagement-ზე ფოკუსირებული Caption

```
შენ ხარ Instagram-ის კონტენტის სპეციალისტი ქართველი აუდიტორიისთვის.

დაწერე 3 ვარიანტი caption-ისა შემდეგი პოსტისთვის:
- პოსტის თემა: {თემა}
- ბრენდის ტონი: {მაგ. მეგობრული, პროფესიული, იუმორისტული}
- აუდიტორია: {მაგ. 25-35 წლის ქალები, ახალგაზრდა დეველოპერები}

თითოეული ვარიანტი უნდა:
- იწყებოდეს ყურადღების მიმპყრობი წინადადებით (hook)
- შეიცავდეს კითხვას, რომელიც კომენტარებს წაახალისებს
- იყოს 80-150 სიტყვის შუალედში
- მთავრდებოდეს მოქმედების მოწოდებით (CTA)

გრამატიკულად სუფთა ქართულით.
```

### 2. Storytelling Caption

```
დაწერე Instagram-ის caption ისტორიის ფორმატში შემდეგი პროდუქტისთვის:
- პროდუქტი/სერვისი: {აღწერა}
- მთავარი სარგებელი: {რას იძლევა მომხმარებლისთვის}
- ემოცია, რომელიც უნდა გამოიწვიოს: {მაგ. ნოსტალგია, ინსპირაცია, კომფორტი}

სტრუქტურა:
1. პრობლემა ან ცხოვრებისეული მომენტი
2. შემობრუნების წერტილი
3. გადაწყვეტა (პროდუქტი ბუნებრივად ჩასვი)
4. დასკვნა + CTA

ქართული ბუნებრივი იყოს, არ ჩანდეს რომ AI-მ დაწერა.
```

### 3. Carousel-ის ტექსტი

```
შემიქმენი Instagram-ის carousel პოსტი (10 სლაიდი) თემაზე: {თემა}.

თითოეული სლაიდისთვის მომეცი:
- სათაური (მაქსიმუმ 6 სიტყვა)
- მოკლე ტექსტი (მაქსიმუმ 25 სიტყვა)

სტრუქტურა:
- სლაიდი 1: hook — რატომ უნდა გადაფურცლოს მომხმარებელმა
- სლაიდი 2-9: ღირებული ინფორმაცია, ნაბიჯ-ნაბიჯ
- სლაიდი 10: CTA + save/share-ზე მოწოდება

caption-იც დაამატე ცალკე — 100 სიტყვამდე.
```

### 4. Reels-ის Caption

```
დაწერე მოკლე და ეფექტური caption Instagram Reel-ისთვის:
- ვიდეოს თემა: {აღწერა}
- მთავარი message: {რას ვამბობ}

წესები:
- მაქსიმუმ 50 სიტყვა
- პირველი 2 სიტყვა იყოს ყურადღების მიმპყრობი
- დაამატე 5 რელევანტური hashtag

3 ვარიანტი მომეცი — სხვადასხვა ტონით.
```

## TikTok-ის სცენარები

### 5. 30-წამიანი ვიდეოს სცენარი

```
დამიწერე 30-წამიანი TikTok ვიდეოს სცენარი თემაზე: {თემა}.

ფორმატი:
- 0-3 წამი: Hook (რა აიძულებს მაყურებელს არ გადაფურცლოს)
- 3-25 წამი: მთავარი კონტენტი (3-4 ძირითადი წერტილი)
- 25-30 წამი: CTA (follow, comment, ან save)

თითოეული სცენისთვის მიუთითე:
- რა ითქმის (ტექსტი)
- რა გამოჩნდება ეკრანზე (ვიზუალი)
- რა ხმოვანი ეფექტი ან მუსიკა

ქართულად, ბუნებრივი სასაუბრო ენით.
```

### 6. Trend-ზე ადაპტირება

```
ამჟამად TikTok-ზე პოპულარულია შემდეგი ტრენდი: {ტრენდის აღწერა}.

ჩემი niche/ბიზნესი არის: {აღწერა}.

შემოგვათავაზე 5 იდეა, თუ როგორ შემიძლია ამ ტრენდის ადაპტირება ჩემი კონტენტისთვის ბუნებრივად — ისე, რომ არ ჩანდეს ფორსირებული.

თითოეული იდეისთვის მომეცი მოკლე სცენარი (3-4 წინადადება).
```

### 7. ვირუსული Hook-ების გენერაცია

```
დამიწერე 10 განსხვავებული hook (პირველი 3 წამი) TikTok ვიდეოსთვის თემაზე: {თემა}.

თითოეული hook უნდა:
- იყოს მაქსიმუმ 8 სიტყვა
- იწვევდეს ცნობისმოყვარეობას, შოკს, ან აღფრთოვანებას
- ქართულად ბუნებრივად ჟღერდეს

გამოიყენე სხვადასხვა ფსიქოლოგიური ხერხი:
- კითხვა
- წინააღმდეგობა მოლოდინთან
- დაპირება
- პირადი გამოცდილება
- სტატისტიკა
```

## LinkedIn-ის პროფესიული პოსტები

### 8. Thought Leadership პოსტი

```
შენ ხარ LinkedIn-ის კონტენტ-სტრატეგი.

დამიწერე professional პოსტი თემაზე: {თემა}.

ჩემი როლი/ექსპერტიზა: {მაგ. Product Manager 5 წლიანი გამოცდილებით}.

სტრუქტურა:
1. პირველი წინადადება — bold statement ან კითხვა, რომელიც კითხვას აგრძელებინებს
2. პერსონალური მოკლე ისტორია ან გამოცდილება (3-4 წინადადება)
3. 3-5 ღირებული insight bullet-ების სახით (გამოიყენე "—" დეფისი, არა emoji)
4. დასკვნა, რომელიც დისკუსიას იწვევს
5. ღია კითხვა აუდიტორიისადმი

ტონი: პროფესიული, მაგრამ ცოცხალი. არ გამოიყენო hashtag-ები ბოლოში 5-ზე მეტი.
```

### 9. Career-related პოსტი

```
დამიწერე LinkedIn-ის პოსტი ჩემი {მიღწევა/ცვლილება} შესახებ:
- რა მოხდა: {აღწერა}
- რას ვისწავლე: {3-4 lesson}
- ვის შეიძლება გამოადგეს: {აუდიტორია}

წესები:
- იყოს ავთენტური და თავმდაბალი (არ ჩანდეს კვეხნა)
- 200-300 სიტყვა
- დაუმთავრდეს კითხვით ან მოწოდებით
- ქართული გრამატიკა იდეალური უნდა იყოს

დაწერე 2 ვარიანტი — ერთი უფრო პერსონალური, მეორე უფრო ანალიტიკური.
```

### 10. Industry Insight პოსტი

```
დამიწერე LinkedIn-ის პოსტი, სადაც გავაანალიზებ შემდეგ ტენდენციას: {ტენდენცია}.

ჩემი industry: {მაგ. Tech, Marketing, Finance}.

სტრუქტურა:
- რა იცვლება (ფაქტი/მონაცემი)
- რატომ აქვს მნიშვნელობა (ანალიზი)
- რას ნიშნავს ეს პროფესიონალებისთვის (პრაქტიკული)
- ჩემი მოსაზრება (პოზიცია)
- კითხვა აუდიტორიისადმი

ტონი — analytical, არა sales-y. გამოიყენე კონკრეტული მაგალითები, არა ზოგადი ფრაზები.
```

## YouTube-ის სათაურები და აღწერები

```
შემიქმენი 10 ვარიანტი YouTube-ის სათაურისა ვიდეოსთვის თემაზე: {თემა}.

თითოეული სათაური უნდა:
- იყოს მაქსიმუმ 60 სიმბოლო
- შეიცავდეს emotional trigger (ცნობისმოყვარეობა, შიში, აღფრთოვანება)
- არ იყოს clickbait, რომელიც ავარდნას იწვევს

გამოიყენე სხვადასხვა ფორმულა:
- "როგორ ..."
- "X რჩევა, რომელიც ..."
- "რატომ ..."
- "X რამ, რომელიც ყველამ უნდა იცოდეს ..."
- კითხვა-სათაურები

ქართულად, გრამატიკულად სუფთად.
```

### 14. ვიდეოს აღწერის (Description) Template

```
დამიწერე YouTube-ის ვიდეოს სრული აღწერა:
- ვიდეოს თემა: {თემა}
- მთავარი ღირებულება: {რას ისწავლიან}
- ჩემი არხი/ბრენდი: {აღწერა}

სტრუქტურა:
1. პირველი 2 წინადადება — hook (გამოჩნდება search-ში)
2. რას მოიცავს ვიდეო (3-5 ბულეტი)
3. Timestamp-ები (ჩამონათვალი)
4. რესურსები და ლინკები
5. ჩემი სოციალური ქსელები
6. რელევანტური hashtag-ები (5-7)

SEO-ზე ფოკუსი — გასაღები სიტყვები ბუნებრივად ჩაანაცვლე.
```

## ბიზნესი და ბრენდინგი

### 15. ბრენდის Voice Guide

```
შემიქმენი ბრენდის voice guide შემდეგი ბიზნესისთვის:
- ბიზნესი: {აღწერა}
- აუდიტორია: {ვინ}
- ღირებულებები: {3-5 ღირებულება}

შედეგად მომეცი:
1. ბრენდის ტონი — 3 ზედსართავი (მაგ. პროფესიული, თბილი, ინფორმაციული)
```

## დასკვნითი რჩევები

**1. ყოველთვის შეამოწმე გრამატიკა.** AI ქართულზე მუშაობს, მაგრამ არ არის სრულყოფილი. წაიკითხე და გაასწორე.

**2. დაამატე პერსონალური შეხება.** AI გაძლევს კარგ საფუძველს — მაგრამ ნამდვილი ღირებულება შენი გამოცდილებისგან მოდის. ჩასვი პირადი ისტორია, კონკრეტული მაგალითი.

**3. იცადე სხვადასხვა მოდელი.** ChatGPT, Claude და Gemini განსხვავებულად წერენ. ერთი და იგივე prompt-ისთვის სხვადასხვა შედეგი მიიღე და აარჩიე საუკეთესო.

**4. შეინახე შენი საუკეთესო prompt-ები.** დაიწყე საკუთარი prompt-ბიბლიოთეკა — დროთა განმავლობაში ისინი შენი მთავარი აქტივი გახდება.

**5. იცი, სად არის ზღვარი.** AI-ს გამოიყენე როგორც ასისტენტი, არა შემცვლელი. ბრენდის ხმა შენი უნდა იყოს, არა მოდელის.
